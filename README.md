# AWS-EKS-Cluster-with-Terraform-Monitoring
Production-ready and highly available, this project uses Terraform to deploy an **Amazon EKS v1.31 cluster** within a custom VPC. It optimizes costs by mixing **On-Demand and Spot instances** while ensuring security through private subnets and **IRSA**. The setup includes essential add-ons like EBS CSI and VPC CNI for a turn-key solution.
